# Unbounded Project Constitution
## 第七章：世界核心规律（Core Rules）

> "规则不是为了限制世界，而是为了让世界能够无限成长。"

---

## 十六条世界法则

1. **世界先于玩家** — 世界在玩家出生前已存在，离开后仍继续
2. **同样原因必然产生同样结果** — 规则不因玩家身份改变
3. **世界不制造资源** — 木头来自树，铁来自矿，必须有来源
4. **所有生命遵循统一规律** — 都会受伤/死亡/燃烧/中毒
5. **自然不会服务玩家** — 森林不是为木材，兔子不是为食物
6. **复杂行为来自简单规则** — 行为来自规则，不是来自脚本
7. **局部计算** — 无限世界永远不能整体模拟
8. **时间真实存在** — 树成长/天气变化/文明发展/尸体腐烂
9. **数据决定内容** — 程序负责解释，数据负责定义
10. **规则高于例外** — 例外越少，世界越统一
11. **每一种存在都应有存在理由** — 既不影响世界也不影响玩家就不该存在
12. **世界能够解释自己** — 玩家看到任何现象都应该能回答"为什么"
13. **历史来自积累** — 不是剧情，是时间+事件+文明+玩家的共同作用
14. **简单规则可以无限组合** — 火+风+雨+木材=无数可能，没有任何脚本
15. **世界永远保持开放** — 不封闭可能性，不规定唯一玩法
16. **世界不是目标，世界是平台** — 真正稳定的不是内容，而是规则
